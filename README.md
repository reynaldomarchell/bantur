## Installation

1. Clone this repository

```bash
  git https://github.com/reynaldomarchell/bantur.git
```

2.  Go to the project directory

```bash
  cd bantur
```

3. Install the dependencies

```bash
  npm i
```

4. Start the json server

```bash
  npm run server
```

5. Start the app

```bash
  npm run dev
```
